## Descripción del Proyecto

Este proyecto forma parte de la asignatura Laboratorio de Desarrollo de Herramientas (LDH) y se centra en la gestión de proyectos software, integración continua y pruebas unitarias utilizando GitHub y JUnit.

El código base corresponde a **BiCIAM** (Biblioteca de Algoritmos Metaheurísticos), una librería Java para la implementación y experimentación con algoritmos metaheurísticos.
