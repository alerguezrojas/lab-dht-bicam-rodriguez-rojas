Implementa la lógica de aceptación de candidatos para el algoritmo de Recocido Simulado Multicaso. 

Esta clase extiende de \doxylink{classlocal__search_1_1acceptation__type_1_1_acceptable_candidate}{Acceptable\+Candidate} y define las reglas para aceptar o rechazar una solución candidata basándose en criterios de dominancia y probabilidad.
